Print

This page is set up to easily print multiple cards at once. The page is built using values set in the url, or it falls back to defaults if valid values aren't provided. Each card will automatically resize within its bounding box. To print, simply use your browser's print functionality (this information will be hidden).

count
Total number of cards to show. Default: 10
margin
Page margin specified in mm, cm, or in. Default: .5in .75in
width
Width of each card specified in mm, cm, or in. Default: 3.5in
height
Height of each card specified in mm, cm, or in. Default: 2in
